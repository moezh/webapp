# Web Application

A web-based application built with the following stack:

- `Bun` + `Astro` for a `TypeScript`-based web stack
- `PostgreSQL` as the database
- `Docker` for containerization and deployment
- `GitHub Actions` for CI/CD automation

## Prerequisites

- **Docker** (Docker Engine or Docker Desktop) installed and running
- **Bun** installed

## Getting Started

1. To start all services in **development** mode, run the command: `bun run dev`

1. To start all services in **production** mode, run the command: `bun run prod`

1. To stop all running services, run the command: `bun run stop`
